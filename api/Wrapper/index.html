Wrapper

A Wrapper is useful for instantiating and managing instances of a class based off tags.

Call​

</>
Wrapper.Call(
self: Wrapper<T>,
Object: T,
Method: string,
...: any
) → any

Calls a method on the object corresponding with the given instance. You can pass any parameters you'd like after the object and the method name.

Note

Setting Debug to true in the options parameter in the constructor will have this method use debug warns. 

local object = wrapper:GetObject(someInstance)
wrapper:Call(object, "DoSomething")

Apply​

</>
Wrapper.Apply(
self: Wrapper<T>,
Inst: Instance
) → T?

Creates a new object for an instance. It will call the constructor method of the object and give pass in the instance, trove, and optionally a GUID. 

local object = wrapper:Apply(someInstance)
if object then
	object: DoSomething()
end

Revoke​

</>
Wrapper.Revoke(
self: Wrapper<T>,
Inst: Instance
) → ()

Cleans up an object.

Note

This will nullify the object's metatable, so you will not be able to use the object after revoking it from Wrapper. 

local object = wrapper:Apply(someInstance)

-- cleans up the instance in the wrapper
wrapper:Revoke(someInstance)

-- you could also destroy the instance, thereby invoking the tag removed event
someInstance:Destroy -- also cleans up

RevokeAll​

</>
Wrapper.RevokeAll(self: Wrapper<T>) → ()

Destroy​

</>
Wrapper.Destroy(self: Wrapper<T>) → ()

Destroys all tracked objects, cleans the trove object and all signals, rendering this object useless.

Metatable Removal

Running this will make any further attempts to call a method of Wrapper error. 

wrapper:Destroy()
